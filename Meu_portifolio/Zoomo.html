Zoomo

História do Jogo:

Esse jogo foi criado para a realização de uma atividade em sala, na qual nós alunos devíamos fazer um jogo para dispositivos móveis usando alguma mecânica aprendida em sala, eu e meu colega resolvemos fazer com a mecância de Zoom ao pinçar a tela.

Personagens do Jogo:

O único personagem do jogo é o jogador, que é controlado pelo deslize do dedo na tela.

Jogabilidade:

Para progredir na fase o jogador deve pinçar a tela para aumentar o tamanho do personagem, para se movimentar ele deve deslizar o dedo para o lado que você quer que o jogador vá.
